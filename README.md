# Seminarska_OOP
Opis:
To je moja seminarska naloga v C#. Program je aplikacija za upravljanje parkirišča, kjer lahko dodajamo vozila, jih spustimo noter in ven ter izračunamo ceno parkiranja.

Kaj program omogoča:
vnos vozila (registrska, tip, višina)
prihod vozila na parkirišče
izhod vozila iz parkirišča
izračun cene glede na ure
prikaz zasedenosti parkirišča
prepreči vstop, če je parkirišče polno
Kako zagnati
Prenesi projekt iz GitHuba
Odpri v Visual Studio
Zaženi projekt (F5)
Opombe:
registrska ne sme biti prazna
če je parkirišče polno, vstop ni možen
cena se računa glede na čas
